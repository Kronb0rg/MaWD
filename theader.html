Please complete the following tasks


  1. Select the 'Heart'-icon in the header. To complete the Heart-task, press the "Make words"-button and find the three red words in the text. Then select these (in order!) in the dropdown menus at the bottom of the page and press the "Check answers"-button. If correct, proceed to the Lightbulb-task.
  2. Select the 'Lightbulb'-icon in the header. To complete the Lightbulb-task, press the "Make words"-button, enter the generated word into the search bar, and search the text to identify how many times the word appears. Select this number in the dropdown menu at the bottom of the page and press the "Check answers"-button. If correct, proceed to the next step.
  3. Repeat the above tasks 3 times each, making sure to alternate between task 1 and task 2. When done, press the "Header selection"-button on this page and select the other header option. If you have already completed each task 3 times for each header, the test is now complete.
